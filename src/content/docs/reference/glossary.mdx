## GBA Terms

| Term | Definition |
|------|------------|
| **BIOS** | Built-in firmware on the GBA |
| **DMA** | Direct Memory Access - hardware memory copying |
| **EWRAM** | External Work RAM (256 KB) |
| **IWRAM** | Internal Work RAM (32 KB, fast) |
| **OAM** | Object Attribute Memory - sprite control |
| **VBlank** | Vertical blank - safe time for graphics updates |
| **VRAM** | Video RAM - graphics storage |

## Pokemon Terms

| Term | Definition |
|------|------------|
| **EV** | Effort Value - training bonus to stats |
| **IV** | Individual Value - hidden genetics |
| **OT** | Original Trainer |
| **PID** | Personality ID - determines characteristics |
| **STAB** | Same-Type Attack Bonus |
